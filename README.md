# cv-virt-joystick
Testing OpenCV with video. Writing a simple virtual joystick.

***
Required install CMake, OpenCV

<h2> Running: </h2>

```
git clone https://github.com/vadrx/cv-virt-joystick
./build/vidcv ../test/dog1.mp4
./build/vidcv ../test/dog2.mp4
```

<h2> Building: </h2>

```
cd build
cmake ..
make
```
